# Food Ordering Backend (Django)

## 📌 Overview
This is a backend project built using Django to manage food items and order processing.

## 🚀 Features
- View food items (GET API)
- Create orders (POST API)
- View all orders (GET API)

## 🛠️ Tech Stack
- Python
- Django
- SQLite

## 🔗 APIs
- /foods/ → Get all food items
- /order/ → Create new order
- /orders/ → Get all orders

## ▶️ How to Run
1. Clone the repo
2. Run `python manage.py runserver`
3. Open `http://127.0.0.1:8000/`
